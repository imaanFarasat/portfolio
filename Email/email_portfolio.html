Best Practices Checklist

Every email I create follows industry best practices to ensure maximum deliverability, engagement, and compliance. Here's my comprehensive checklist:

Legal & Compliance

  • Unsubscribe link included and visible
  • CAN-SPAM Act compliance
  • GDPR compliance (if applicable)
  • Physical mailing address included
  • Privacy policy link

Accessibility

  • Alt text on all images
  • WCAG AA compliance (4.5:1 contrast)
  • Screen reader compatibility
  • Descriptive link text (not "click here")
  • Minimum 14px font size

Technical Quality

  • HTML validation (W3C compliant)
  • File size under 100KB
  • Inline CSS used
  • Table-based layout structure
  • Media queries for responsive design

Content & Design

  • Clear, prominent call-to-action
  • Mobile-responsive design
  • Dark mode support
  • Plain text version available
  • Preheader text optimized

Images & Media

  • Images optimized and compressed
  • JPG/PNG format (no AVIF/WEBP)
  • Appropriate image dimensions
  • Alt text for all images
  • Image fallbacks for blocked images

Deliverability

  • Spam score checked and optimized
  • Avoid spam trigger words
  • Proper sender authentication (SPF, DKIM)
  • Clean HTML code
  • Text-to-image ratio balanced

Testing & Quality Assurance

  • Tested across major email clients
  • Mobile device testing
  • Dark mode testing
  • Link testing (all links work)
  • Spell check completed

Personalization & Segmentation

  • Personalization tokens tested
  • Dynamic content implemented
  • Segmentation strategy applied
  • Merge tags properly formatted
  • Fallback content for missing data

Note: This checklist is applied to every email template I create. Quality and compliance are non-negotiable in my workflow.
